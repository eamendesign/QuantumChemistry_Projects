# Quantum Chemistry From Scratch

A minimal collection of **Python scripts and notebooks that implement key concepts of quantum chemistry from scratch**.

The goal of this project is to demonstrate how complex theories in **quantum mechanics, electronic structure, and spectroscopy** can be understood through **simple, transparent code**.

Instead of relying on large libraries, most examples are written with only **NumPy + basic Python**, making the underlying physics and algorithms easy to follow.

## Topics

- Hydrogen atom and Rydberg spectrum  
- Wave–particle duality and quantum tunneling  
- Harmonic oscillator and Schrödinger equation  
- Hartree–Fock method  
- Density Functional Theory (1D model)  
- Molecular orbitals and hybridization  
- Reciprocal space and Brillouin zones  
- Computational spectroscopy (IR, X-ray scattering)
